# VidhyaPath 🎓

> AI-powered adaptive learning for Indian students — CBSE, ICSE & State Boards, Classes 1–10.

VidhyaPath personalises every student's study journey by tracking their strengths and weak concepts in real time, then generating board-aligned lessons, notes, and quizzes that adapt to *them* — not the class average. Teachers get a live dashboard with per-student progress, quiz history, and monthly reports.

---

## Features

### For Students
- **Personalised Learning Path** — AI generates a topic order based on the student's grade, board, goals, and self-assessed level. The path auto-updates when 3+ topics turn weak.
- **Adaptive Quizzes** — Every quiz is dynamically generated. On retries, questions focus specifically on the concepts the student got wrong last time.
- **AI Lesson & Notes** — Full chapter lessons and concise notes generated by Google Gemini 2.0 Flash, scoped to the student's exact board and grade.
- **AI Chatbot Tutor** — Ask any syllabus question in English or Hindi. Falls back to a smart offline engine when the API is unavailable.
- **Offline-First Architecture** — Lessons, notes, and quizzes are cached locally. Quiz results queue offline and sync automatically when the connection returns.
- **Hindi & English UI** — Full bilingual interface switchable at any time.
- **PWA / Installable** — Works on mobile, tablet, and desktop. Can be installed as a home-screen app.

### For Teachers
- **Teacher Dashboard** — View all students in your class at a glance: progress %, streak, last active date, weak areas, and recent scores.
- **Per-Student Quiz History** — Full attempt log for every quiz a student has taken, including score, status, and weak concepts per attempt.
- **Monthly Progress Report** — Month-by-month breakdown of attempts, average score, topics covered, and strong vs weak topic counts.
- **Syllabus Management** — Add or remove chapters and topics for individual students to create a customised syllabus.
- **Inactivity Alerts** — Students inactive for 5+ days are flagged automatically.
- **Class Codes** — Create a class, share the code, and students self-enrol. Multiple classes per school supported.

| Layer | Technology |
|-------|-----------|

## Local Setup

### Prerequisites
- Node.js 18+
- An [OpenRouter](https://openrouter.ai) API key (free tier works — starts with `sk-or-`)

### 1. Clone & install

```bash
git clone https://github.com/your-username/VidhyaPath.git
cd VidhyaPath
npm run install:all
```

### 2. Configure environment

Create `server/.env`:

```env
OPENROUTER_API_KEY=sk-or-your_key_here
OPENROUTER_MODEL=google/gemini-2.0-flash-001
```

> The app works fully without an API key using the built-in offline AI engine.

### 3. Run

```bash
# From project root — starts both server and client concurrently
npm run dev
```

- API: `http://localhost:3001`
- UI: `http://localhost:5173`

## Deployment (Vercel)

1. Push to GitHub and import the repo at [vercel.com/new](https://vercel.com/new).
2. Add environment variables in **Project Settings → Environment Variables**:

   | Key | Value |
   |-----|-------|
   | `OPENROUTER_API_KEY` | `sk-or-your_key_here` |
   | `OPENROUTER_MODEL` | `google/gemini-2.0-flash-001` |

3. Deploy. The `vercel.json` handles routing — no extra settings needed.

> **Note:** Vercel's serverless filesystem is ephemeral — the SQLite DB resets on cold starts. For persistent production data, migrate to [Turso](https://turso.tech) (SQLite-compatible, free tier).

## API Reference

### Auth
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/auth/signup` | Student signup |
| POST | `/api/auth/login` | Student login |
| POST | `/api/auth/teacher/signup` | Teacher signup |
| POST | `/api/auth/teacher/login` | Teacher login |

### Students
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/students/:id` | Get student + path + progress |
| POST | `/api/students` | Create / update student profile |

### Learning Path
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/path/generate` | Generate initial learning path |
| POST | `/api/path/update` | Regenerate path based on weak areas |

### Quiz
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/quiz/generate` | Generate quiz for a topic |
| POST | `/api/quiz/submit` | Submit answers, save result |
| GET | `/api/quiz/weakness/:student_id/:topic` | Get weak concepts for a topic |

### Content
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/content/lesson` | Generate lesson content |
| POST | `/api/content/notes` | Generate study notes |
| POST | `/api/content/quiz` | Generate content quiz |

### Teacher
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/teacher/students` | All students in class |
| GET | `/api/teacher/student/:id/quiz-history` | Full quiz history |
| GET | `/api/teacher/student/:id/monthly-report` | Monthly progress report |
| POST | `/api/teacher/add-module` | Add syllabus module to student |
| POST | `/api/teacher/remove-module` | Remove syllabus module |

### Chat & Syllabus
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/chat/message` | Send message to AI tutor |
| GET | `/api/student/syllabus?studentId=X` | Get student's syllabus |
